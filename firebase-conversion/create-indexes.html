Create Missing Firestore Indexes

Manual Step Required:

Firestore requires composite indexes for complex queries. Click the link below to automatically create the missing index:

Create Index in Firebase Console
Instructions:
  1. Click the button above to open Firebase Console
  2. Make sure you're logged in to the correct Google account
  3. Click "Create Index" when prompted
  4. Wait for the index creation to complete (may take a few minutes)
  5. Return here and test the location page
Alternative: Manual Index Creation

If the automatic link doesn't work, create this index manually in Firebase Console:

Collection Group: locations
Fields:
  1. sortOrder (Ascending)
  2. name (Ascending)
  3. __name__ (Ascending)
